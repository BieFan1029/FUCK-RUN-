# FUCK-RUN

基于 [uy-li/runhook](https://github.com/uy-li/runhook) 二次开发

LSPosed 校园跑步传感器仿真模块，支持多款校园跑步 App。

## 支持的 App

| App 名称 | 包名 |
|----------|------|
| 闪动校园 | `com.huachenjie.shandong_school` |
| 闪动校园PRO | `com.huachenjie.shandong_school_pro` |
| 体适能 | `com.bxkj.student` |
| 运动世界校园 | `com.zjwh.android_wh_physicalfitness` |
| 宥马运动 | `android.youma.com` |

## 测试情况

目前只在 OnePlus ACE 5 Pro 上测试通过。

## 使用方法

1. 安装 LSPosed 框架
2. 安装本模块 APK
3. 在 LSPosed 中勾选对应的校园跑步 App
4. 重启手机
5. 打开校园跑步 App → 开始跑步 → 开启 FUCK-RUN 悬浮窗开关
