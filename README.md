# RunHook 步频引擎 (闪动校园适配版)

基于 [uy-li/runhook](https://github.com/uy-li/runhook) 二次开发

在runhook的基础上适配了闪动校园和闪动校园Pro /Based on RunHook, we have adapted it for Shandong Campus and Shandong Campus Pro

目前只在黑厂手机上进行过测试 测试机型：一加ACE5Pro可以正常使用

## 功能介绍

LSPosed 传感器仿真模块，用于模拟跑步步数数据。

## 二改内容

- ✅ 添加闪动校园 (`com.huachenjie.shandong_school`) 包名支持
- ✅ 添加闪动校园PRO (`com.huachenjie.shandong_school_pro`) 包名支持
- ✅ 添加加速度传感器调试日志
- ✅ 优化 LSPosed 作用域配置

## 使用方法

1. 安装 LSPosed 框架
2. 安装本模块 APK
3. 在 LSPosed 中勾选闪动校园/闪动校园PRO
4. 重启手机
5. 打开闪动校园 → 开始跑步 → 开启 RunHook 悬浮窗开关
